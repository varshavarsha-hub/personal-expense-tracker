# Personal Expense Tracker

A modern and user-friendly Personal Expense Tracker built using Flask, SQLite, HTML, CSS, and JavaScript.

This application helps users track their daily expenses, categorize spending, view monthly reports, analyze expenses through charts, and export reports.

---

## Features

### Expense Management
- Add expenses
- Edit expenses
- Delete expenses
- View all expenses

### Expense Details
- Expense title
- Amount
- Category
- Custom date selection

### Dashboard
- Total expense calculation
- Total records count
- Expense summary cards
- Responsive dashboard layout

### Search Functionality
- Search expenses instantly
- Filter expenses dynamically

### Charts & Visualization
- Dynamic expense chart
- Category-wise expense analysis
- Database-driven chart generation

### Monthly Summary
- Monthly expense tracking
- Month-wise spending report

### Export Reports
- Export expense reports to PDF

### User Interface
- Modern UI design
- Responsive layout
- Dark mode support
- Colorful dashboard cards
